## 2. 비용 절약 핵심: 3-Layer 캐싱

```
사용자 요청
    │
    ▼
[Layer 1] Next.js ISR — 페이지 정적 빌드, 1시간마다 재생성 (API 호출 0)
    │
    ▼ MISS
[Layer 2] Redis (Upstash) — 번역 결과 24시간 캐싱
    │
    ▼ MISS
[Layer 3] Supabase DB — 이전 번역 결과 영구 저장
    │
    ▼ MISS (최초 요청만)
[Gemini API] → 번역 → DB 저장 + Redis 캐싱
```

**효과**: 같은 레포 100명이 조회해도 Gemini API는 **1번만 호출**
